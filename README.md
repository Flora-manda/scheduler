# Interview Scheduler

This React application allows users to book, edit and cancel interviews. It combines a concise API with a WebSocket server to build a realtime experience.

## Setup

Install dependencies with `npm install`.

## Dependencies

- axios
- classnames
- normalize.css
- react
- react-dom
- react-scripts
